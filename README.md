# 今天是酒蒙子吗

这是一个由 Codex 辅助开发的微信小游戏 Demo，用于快速验证“醉酒失控 + 数值生存”玩法原型，demo主题为一场派对喝酒。

## 游戏背景

你在一场逐渐失控的聚会上，需要在 5 分钟内做选择：
- 继续喝酒、释放情绪
- 通过手机、对话做出更夸张的社交行为
- 或尽量维持体面并安全离场

随着酒精浓度上升，画面和操作会逐步失真，社交容忍度也会持续承压。

## 主要功能

- 核心属性系统：心情、酒精、容忍、积分、羞耻
- 属性条可视化：顶部条状 UI + 5 分钟倒计时可视化
- 主场景互动：喝酒、假装正常、NPC 对话
- 背景 NPC 三态表情图：随容忍度切换素材
- 手机大界面玩法：
  - 短视频（拖动音量条 1~5 档）
  - 微信骚扰（限时狂点）
  - 朋友圈轰炸（限时狂点）
- 离场平衡小游戏：酒精越高，失衡越快，控制更难
- 多结局与局后海报

## 技术栈

- 微信小游戏（Mini Game）
- TypeScript
- Canvas 2D
- 本地存档（最高分与称号）

## 本地运行

```bash
npm install
npm run build
```

然后使用微信开发者工具按“小游戏项目”方式打开本目录。
